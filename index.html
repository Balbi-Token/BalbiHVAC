          Cálculos e Fórmulas
          
      
            
        
          • Vazão (Recirculação): Q = max((q * L) / (1000 * cp * ΔT) * 3600, V * n)
          
            • Onde: q = Perda térmica (W/m), L = Comprimento do anel (m), cp = Calor específico (J/(kg·K)), ΔT = Variação de temperatura (°C), V = Volume do tanque (m³), n = Taxa de renovação (h⁻¹).
              • 
            
            • Fonte: ASHRAE Handbook - HVAC Systems and Equipment.
            
        
            • 
        
          • Vazão (Consumo): Q = (P * 1000 * S) / (1000 * cp * ΔT) * 3600
          
            • Onde: P = Carga térmica (kW), S = Fator de simultaneidade, cp = Calor específico (J/(kg·K)), ΔT = Variação de temperatura (°C).
              • 
            
            • Fonte: ABNT NBR 16401-1.
            
        
            • 
        
          • Altura Manométrica Total: Ht = Hs + hf + hm + he
          
            • Onde: Hs = Altura estática (m), hf = Perda por fricção (m), hm = Perdas menores (m), he = Queda em equipamentos (m).
              • 
            
            • Fonte: Darcy-Weisbach Equation, ASHRAE.
            
        
            • 
        
          • Potência: P = (ρ * g * Q * Ht) / (η * 1000)
          
            • Onde: ρ = Densidade (kg/m³), g = Aceleração gravitacional (m/s²), Q = Vazão (m³/s), Ht = Altura total (m), η = Eficiência.
              • 
            
            • Fonte: ABNT NBR 11381.
            
        
            • 
        
          • NPSHa: NPSHa = (Patm / (ρ * g)) + hs - (Pv / (ρ * g)) - hfs - (vs² / (2 * g))
          
            • Onde: Patm = Pressão atmosférica (Pa), hs = Altura de sucção (m), Pv = Pressão de vapor (Pa), hfs = Perda por fricção na sucção (m), vs = Velocidade na sucção (m/s).
              • 
            
            • Fonte: ASHRAE Handbook - Pumping Systems.
